  Modifier mappings (QMK → ZMK):
  - S() → LS() (shift layer: digits, #, ", ¨, >, ?, !, …, =, etc.)
  - ALGR() → RA() (AltGr layer: æ, €, {, }, ç, |, etc.)
  - S(ALGR()) → LS(RA()) (Shift+AltGr layer: ™, —, ≠, ≥, etc.)

  AZERTY-remapped letters are preserved: A↔Q, Z↔W, M→SEMI, .→M matching the French AFNOR NF Z71-300 standard layout.
